The Zenith Responsive Multipurpose HTML HTML5 website template is a custom design package that is exclusively made for NFT Portfolio and Marketplace. It provides the best solution for discovering best Digital art and Collecting NFTs where you can create and sell your NFTs. It has a modern, clean, creative & unique design based on the latest technology..

It contains About layouts and Wallet layout, checkout layout, contact layout and 15+ valid HTML5 page templates and different blog page designs. Folder structure of this template is as follows:

  • assets - contains all assets of this template
    • css - contains all vendor and main style css files
    • sass - contains all sass files.
    • fonts - contains all font files.
    • images - contains all images used in the template
    • js - contains all vendor and custom js files
  • index.html - Home Page
  • collection.html - Mens Collection Page
  • singleproduct.html - Single Product Page
  • checkout.html - Checkout Page
  • wishlist.html - Wishlist Page

HTML Structure

This template is based on a responsive and full screen layout. And it's built using HTML5 standards.

index.html

From beginning of the <body> tag till the ending </body> tag, all the codes and texts are used for the template structure. You can change any text from the template by navigating through the index.html file and localizing each line of text you want to change.

Like index.html, you can change any page included with this template starting from <body> tag till ending </body> tag.

Change Images

When you download the template files you will see the highlights, New Arrivals, Blogs, banners, slider and Products list and its images. Instead of those originals, you can change or replace with your desired images. you can Go to image section you can see like this- img src="assets/images/tshirt-white.jpg" . Here you can replace your image url.

Icons

We are using fontawesome icons, you can use any types of svg and fontawesome icon and google icon on that template.

CSS Files

All the css files are located in the assets/css folder of the template.
style.css - Main stylesheet which includes all the styles of the template.
responsive-style.css - Responsive stylesheet which includes all the responsive styles of the template.
You can edit style easily, add your style bottom of the swiper-bundle.min.css and see the changes

JavaScript Files

All 3rd party plugin and our custom JavaScript files are located in the js folder.
You can add your custom js codes there if you need to.

Note : Please DO NOT edit this files unless you have JavaScript coding experience. Simple mistakes can cause serious damage to the scripts.
